# ✋ FiveMe

FiveMe is a social high-five tracking Android application that brings real-world positivity into the digital space!

## ✨ Features

- **Give & Receive High-Fives**: Acknowledge people in the real world and track it digitally.
- **Proximity Discovery**: Automatically discovers nearby FiveMe users.
- **Inspiration Chains**: Track the ripple effect of positivity! See how a single high-five inspires others to pass it on through an interactive visual ancestry chain.
- **Vibrant UI**: Built entirely in Jetpack Compose with a sleek, neon-dark theme.
- **Mock Sandbox Simulator**: Quickly create mock nearby users and simulate inbound high-fives directly from the app for testing.

## 🛠 Tech Stack

- **Kotlin 2.0.21 & KSP**
- **Jetpack Compose** (Material 3)
- **Room Database** (Local Storage)
- **Jetpack Navigation 3**

## 🚀 Getting Started

1. Clone or download the repository.
2. Open the project in Android Studio.
3. Run the app on an Android Emulator or physical device.
4. Tap the **Sandbox** (beaker icon) to simulate nearby users and incoming high-fives!
